# LifeLink - Blood Donation System

LifeLink is a modern web application designed to bridge the gap between blood requesters and donors. It provides real-time notifications, chat functionality, and a seamless flow for managing blood requests.

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, Socket.io-client
- **Backend**: Node.js, Express, MongoDB, Socket.io, Nodemailer
- **Authentication**: JWT, Google OAuth 2.0, OTP-based email verification

## Features
- Real-time global blood request alerts.
- Targeted email notifications for matched donors.
- Live chat between requesters and donors.
- User profile management and donation history tracking.
- Secure OTP verification for registration and password recovery.
